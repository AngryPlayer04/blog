Olá a todos, meu nome é Cássio#

Decidi criar este blog como forma de documentar minha tragetória na tecnologia e principalmente sobre homelab/selfhost, além de ajudar outros que possam precisar.

Tenho em casa um servidor com TrueNas Scale rodando num AMD Athlon A800 (um dual core que dá conta do recado) com 16Gb de RAM e dois HDs de 1TB em mirror.

Tenho rodando:

  • Adguard Home - Bloqueio de anúncios via DNS;
  • Jellyfin - Servidor de filmes e séries;
  • Homarr - Homepage;
  • Navidrome - Servidor de música;
  • Wireguard - VPN para acessar remotamente meu servidor;
  • Gitea - Servidor Git para backups;
  • Syncthing - Programa para sincronização de arquivos.

Devo demorar pra postar novamente, já que mexo nele uma vez ou outra.
